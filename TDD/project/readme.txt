project.toml - ConfigFile

app   - пакет с основным управляющим кодом
io    - пакет для операция ввода/вывода
utils - вспомогательный пакет 

пакет и каталог отличаются тем что 
для пакета создается собственное пространство имён.
если в каталог поместить файл __init__.py,
то интерпритатор будет воспринимать его как пакет
и будет создавать для него пространство имен
